MATLAB

MATLAB для пользователей Excel – русскоязычный web-семинар по продуктам компании MathWorks

Компания Softline (http://www.softline.ru) предлагает Вашему вниманию записанный вебинар на русском языке, посвященный основным возможностям продуктов MATLAB для пользователей Excel.

Многие технические специалисты сталкиваются с ограничениями, используя Excel в приложениях для анализа своих данных. Этот сетевой семинар рассказывает о том, как можно дополнить стандартные возможности Excel с помощью MATLAB, обеспечив доступ к тысячам предварительно созданных функций инженерного и расширенного анализа, а также универсальных инструментов визуализации. Кроме того, на сетевом семинаре будет обсуждаться, как MATLAB повышает скорость расчетов и позволяет обрабатывать большие объемы данных.

Во время демонстрации продукта освещаются следующие темы.

  • Обмен данными между MATLAB и Excel.
  • Создание пользовательских функций анализа и визуализации в MATLAB и вызов их из Excel.
  • Автоматическое создание отчетов для документирования результатов выполненного анализа.
  • Развертывание собственных функций MATLAB в качестве надстроек Excel.

Для этого сетевого семинара не требуется предварительного знакомства со средой MATLAB.

Вебинар доступен на странице сайта компании MathWorks.
